412Connect

Project Description

412Connect aims to showcase organizations from marginalized communities to local university students through an online scavenger hunt. The CMU IS student team primarily worked on addressing the current pain points of the website by redesigning the user dashboard, revamping the user authentication process, and remodeling user feedback collection. These changes should create a more intuitive user experience that helps the platform better retain students down the line.
